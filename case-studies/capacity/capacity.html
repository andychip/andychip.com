User Experience, User Interface Design, in-browser design

{{page.title}}

Previously known as theZooApp, Capacity is an app for Zoos, Aquariums, Parks and attractions. It was rebranded in 2014, and I undertook content analysis (subsequently restructuring the site) and the UI Design. This was done in-browser to speed up completion.

UI was designed and built in-browser.

  • Bink - iOS UX and UI Design
  • Plated - Optimising mobile checkout flow and UI Design
  • Prove - Identity and UI design
  • Coburg Banks - IA and wireframes
  • Love Worth Finding - IA, wireframes and in-browser design & build.
