Bink iOS App - MVP

Bink is an iOS app helping creatives gain greater exposure, and giving recruiters the chance to find unknown talent. It enables different kinds of creatives to connect multiple content sources to one account.

This project was started from scratch, going through content analysis, sign up flow, red routes, wireframes, userflows and UI before I put together a prototype for the client to use for investor pitches. It also took in some branding and web work
